JavaScript
(

    JavaScript is a high-level, interpreted programming language that
    is widely used to make web pages interactive and dynamic.


    -event: anything that the user does

    -querySelector: look trough an html page an extract a element


     // Eventos de Mouse:

    click: Disparado quando o usuário clica em um elemento.
    dblclick: Disparado quando o usuário faz um duplo clique em um elemento.
    mouseover: Disparado quando o ponteiro do mouse passa sobre um elemento.
    mouseout: Disparado quando o ponteiro do mouse sai de um elemento.
    mousedown: Disparado quando o botão do mouse é pressionado.
    mouseup: Disparado quando o botão do mouse é liberado.
    mousemove: Disparado quando o ponteiro do mouse se move dentro de um elemento.

Eventos de Teclado:

    keydown: Disparado quando uma tecla é pressionada.
    keyup: Disparado quando uma tecla é liberada.
    keypress: Disparado quando uma tecla é pressionada e liberada (não é recomendado para novos projetos, use keydown e keyup).

Eventos de Formulário:

    submit: Disparado quando um formulário é enviado.
    change: Disparado quando o valor de um elemento <input>, <select>, ou <textarea> muda.
    focus: Disparado quando um elemento ganha foco.
    blur: Disparado quando um elemento perde o foco.

Eventos de Janela:

    load: Disparado quando a página e todos os seus recursos (imagens, scripts, etc.) são totalmente carregados.
    resize: Disparado quando a janela do navegador é redimensionada.
    scroll: Disparado quando a página ou um elemento com barra de rolagem é rolado.
    unload: Disparado quando o usuário está prestes a sair da página.

Outros Eventos Comuns:

    contextmenu: Disparado quando o botão direito do mouse é clicado, geralmente para abrir um menu de contexto.
    input: Disparado cada vez que o valor de um <input> ou <textarea> muda.
    wheel: Disparado quando o usuário gira a roda do mouse.
    drag: Disparado enquanto um elemento está sendo arrastado.




)
